# Rotom Editor Extensions

Editor support for the [Rotom](https://github.com/KalaayPT/rotom) scripting language — a compiler toolchain for Pokémon Generation 4 field scripts.

## Contents

| Package | Purpose |
|---------|---------|
| `editors/vscode/` | VS Code extension (LSP client + TextMate grammar) |
| `editors/zed/` | Zed extension (tree-sitter grammar + LSP launcher) |
| `editors/neovim/` | Neovim Lua plugin (tree-sitter + LSP setup) |
